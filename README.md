# AWESOME WS

Get ready for Awesome ws chat

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install packages.

```bash
npm install
```

## Usage
Start server
```bash
npm run server
```
Start client
```bash
npm run client
```
Start tests
```bash
npm run watch
```
